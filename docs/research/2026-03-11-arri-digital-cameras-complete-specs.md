## Findings by Camera

### Finding 1: Arriflex D-20 (2005)

- **Confidence**: HIGH
- **Sensor type**: Super 35, CMOS, Bayer pattern
- **Sensor size (total)**: 23.76 x 17.82 mm (4:3 full sensor area; uses 23.76 x 13.365 mm in 16:9 HD mode)
- **Native resolution**: 2880 x 2160 (Data mode, 4:3); 2880 x 1620 (HD mode, 16:9); downsampled to 1920 x 1080 HD output
- **Lens mount**: PL (54mm, Super 35 centered)
- **Native aspect ratios**: 4:3, 16:9
- **Frame rates**: 1-60 fps
- **Recording**: 12-bit ARRIRAW, 10-bit HD-SDI (4:4:4 RGB, 4:2:2 YCbCr)
- **Notes**: ARRI's first digital cinema camera. Originally announced 2003, available 2005.
- **Supporting sources**:
  - VFX Camera Database - Arriflex D-20 (https://vfxcamdb.com/arri-arriflex-d-20/) -- sensor: 23.760 x 13.365 mm (16:9), 2880x2160/2880x1620
  - Wikipedia - Arriflex D-20 (https://en.wikipedia.org/wiki/Arriflex_D-20) -- 1 to 60 fps, PL mount, Super 35 CMOS sensor
  - Cameraquip (http://www.cameraquip.com.au/rentals/digital-cameras/arriflex-d-21/) -- D-21 specs (largely shared with D-20)

### Finding 3: ALEXA Classic / ALEXA EV (2010)

- **Confidence**: HIGH
- **Sensor type**: Super 35, ALEV III CMOS, Bayer pattern, rolling shutter
- **Sensor size (full)**: 28.17 x 18.13 mm (total photosites: ~3392 x 2200)
- **Sensor size (16:9 active)**: 23.76 x 13.37 mm
- **Native resolution (16:9)**: 2880 x 1620 (2.8K) -- this is the native recording resolution
- **Supported resolutions**: 2880 x 1620 (2.8K ARRIRAW), 1920 x 1080 (HD ProRes), 2048 x 1152 (2K)
- **Lens mount**: PL (with LDS on Plus models); EF mount available as swap
- **Native aspect ratios**: 16:9 (standard models); Open Gate NOT supported on Classic
- **Frame rates**: 0.75-60 fps standard; 0.75-120 fps with High Speed license
- **Dynamic range**: 14+ stops
- **Base sensitivity**: EI 800 (adjustable EI 160-3200)
- **Recording**: 12-bit ARRIRAW (via external recorder or XR module), Apple ProRes (422, 422 HQ, 4444), DNxHD
- **Notes**: The "Classic" designation was retroactive after XT series launched. The ALEXA EV is the electronic viewfinder variant. The standard ALEXA and ALEXA Plus are 16:9 sensor mode only (no 4:3).
- **Supporting sources**:
  - ARRI ALEV Sensors page (https://www.arri.com/en/learn-help/arri-camera-technology/alev-sensors) -- ALEV III, Super 35, same as 35mm film frame
  - VFX Camera Database - ALEXA XT (https://vfxcamdb.com/arri-alexa-xt/) -- identical sensor specs to Classic, 28.25 x 18.17 mm Open Gate
  - Newsshooter history article (https://www.newsshooter.com/2018/02/02/a-look-back-at-the-history-of-the-arri-alexa/) -- release date, model variants

### Finding 6: ALEXA Studio (2012)

### Finding 8: ALEXA XT Family (2013) -- XT, XT Plus, XT M, XT Studio

- **Confidence**: HIGH
- **Sensor type**: Super 35, ALEV III CMOS, Bayer pattern, rolling shutter (XT Studio: mechanical shutter option)
- **Sensor size (Open Gate)**: 28.25 x 18.17 mm (3424 x 2202 stored; 3414 x 2198 active)
- **Sensor size (4:3)**: 23.76 x 17.82 mm (2880 x 2160)
- **Sensor size (16:9 2.8K)**: 23.76 x 13.37 mm (2880 x 1620)
- **Sensor size (16:9 3.2K)**: 26.14 x 14.70 mm (3164 x 1778)
- **Sensor size (16:9 HD)**: 23.76 x 13.37 mm (1920 x 1080)
- **Supported resolutions**:
  - Open Gate: 3424 x 2202 (3.4K) -- XT/XT Plus only, NOT Classic
  - 4:3: 2880 x 2160
  - 16:9: 3164 x 1778 (3.2K), 2880 x 1620 (2.8K), 2048 x 1152 (2K), 1920 x 1080 (HD)
- **Lens mount**: PL with LDS (all variants)
- **Native aspect ratios**: Open Gate (~1.55:1), 4:3, 16:9
- **Frame rates**: 0.75-120 fps (16:9), up to 96 fps (4:3, 6:5), up to 90 fps (Open Gate)
- **Dynamic range**: 14+ stops
- **Recording**: Built-in XR Module for in-camera ARRIRAW; Apple ProRes (422 HQ, 4444, 4444 XQ); media: XR Capture Drives, SxS PRO+, CFast 2.0
- **Variants**:
  - **XT**: Base model with XR module for in-camera ARRIRAW
  - **XT Plus**: Adds 4:3 sensor mode, wireless control, extended electronics
  - **XT M**: Split-body design (like ALEXA M but with XR module)
  - **XT Studio**: 4:3 sensor, optical viewfinder, mechanical mirror shutter
  - **XT Plus B+W**: Monochrome sensor variant, rental only (January 2014)
- **Notes**: The XT series added Open Gate sensor mode (not available on Classic). The XR module enabled in-camera ARRIRAW recording without external recorder.
- **Supporting sources**:
  - VFX Camera Database - ALEXA XT (https://vfxcamdb.com/arri-alexa-xt/) -- all sensor dimensions per mode

## Contradictions

- **Alexa Classic/XT sensor total size: 28.17 vs 28.25 mm**: One ARRI document references the total sensor as 28.17 x 18.13 mm; the VFX Camera Database and ARRI Mini recording areas document cite 28.25 x 18.17 mm for Open Gate. The difference appears to be recording area vs total photosite area. The XT/Mini Open Gate stored resolution is 3424 x 2202 within a total photosite count of ~3424 x 2202 (the active recording area may differ slightly from total sensor area).
  - ARRI recording areas document: 28.25 x 18.17 mm (https://www.arri.com/resource/blob/178064/d84031615b6bf09446aeb7972066b129/alexa-mini-recording-areas-surround-views-and-framelines-sup-4-0-data.pdf)
  - Various sources citing 28.17 x 18.13 mm as "original sensor size"

- **ALEXA 65 mount: XPL vs LPL**: The ALEXA 65 was originally equipped with the XPL mount (72mm diameter) at launch in 2015. The current ARRI Rental page states it uses LPL mount. The camera was retrofitted with LPL after 2018.
  - ARRI Lens Mounts page (https://www.arri.com/en/learn-help/arri-camera-technology/lens-mounts-and-lds-2) -- XPL introduced 2015
  - ARRI Rental A65 page (https://www.arrirental.com/en/cameras/exclusive-cameras/alexa-65) -- currently lists LPL

- **ALEXA 265 frame rate maximums**: The Newsshooter source lists specific maximums per mode (29/37/43 fps), while the ARRI Rental page lists 0.75-60 fps overall. These may differ based on Codex Compact Drive generation vs SXR drives.
  - Newsshooter (https://www.newsshooter.com/2024/12/05/arri-alexa-265/) -- 29/37/43 fps by mode
  - ARRI Rental page (https://www.arrirental.com/en/cameras/digital-65-mm-cameras/alexa-265) -- 0.75-60 fps overall
