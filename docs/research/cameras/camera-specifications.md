## ADAPTER / LENS MOUNT NOTES

### Micro Four Thirds (MFT) Adapters
All MFT-mount cameras (Panasonic GH series, BGH1, Blackmagic BMPCC original/4K, Studio Cameras 4K) can use:
- Native MFT lenses
- Canon EF/EF-S lenses via Metabones/Viltrox Speed Booster or passive adapters
- Nikon F lenses via passive adapters
- PL lenses via MTF/Wooden Camera adapters
- C-mount lenses via adapter

### Canon EF Mount Adapters
Cameras with EF mount (Blackmagic 6K series, URSA Broadcast G2, EVA1, VariCam LT) can use:
- Native Canon EF/EF-S lenses
- PL lenses via PL-to-EF adapter (some with electronic pass-through)
- Sigma/Tamron/etc. EF-mount lenses

### L-Mount Adapters
L-Mount cameras (Panasonic S series, Blackmagic Cinema Camera 6K, PYXIS 6K L) can use:
- Native L-Mount lenses (Panasonic, Sigma, Leica)
- Canon EF lenses via Sigma MC-21 adapter
- Canon RF lenses via third-party adapters (limited)
- Leica M lenses via adapter
- PL lenses via adapter

### Fujifilm X Mount Adapters
All Fujifilm X-series cameras can use:
- Native Fujifilm X Mount lenses
- Canon EF lenses via Fringer/Viltrox adapters (with AF)
- Leica M lenses via adapter
- Nikon F lenses via passive adapter
- PL lenses via adapter

### Fujifilm G Mount Adapters
All GFX cameras can use:
- Native Fujifilm G Mount lenses
- Hasselblad H lenses via Fujifilm H Mount Adapter
- Hasselblad V lenses via adapter
- Mamiya 645 lenses via adapter
- Fujifilm X Mount lenses via adapter (with crop)
- Canon EF lenses via Fringer adapter (with AF)

### PL Mount Adapters
PL-mount cameras (VariCam 35, VariCam Pure, URSA Mini Pro 12K, URSA Cine 12K) can use:
- Native PL cinema lenses
- EF lenses via PL-to-EF adapter/mount swap

## CONTRADICTIONS

- **Blackmagic URSA Broadcast sensor size labeling**: Blackmagic markets the URSA Broadcast as having a "2/3 inch sensor" for B4 lens compatibility, but the actual physical sensor measures 23.10 x 12.99 mm -- much larger than a true 2/3" sensor (8.8 x 6.6 mm). The camera uses a windowed/cropped area of the sensor when B4 lenses are attached.
  - Source: [Blackmagic URSA Broadcast Tech Specs](https://www.blackmagicdesign.com/products/blackmagicursabroadcast/techspecs) says "2/3 inch sensor size when using 4K B4 mount (Actual sensor size 23.10mm x 12.99mm)"

- **Panasonic S1R sensor size**: Apotelyt lists 36.0 x 24.0 mm while some sources reference the more common 35.6 x 23.8 mm used by the S1/S1H. The S1R does appear to have a slightly larger sensor than the S1/S1H.
  - Source A: [Apotelyt S1R](https://apotelyt.com/camera-specs/panasonic-s1r-sensor-size) -- 36.0 x 24.0 mm
  - Source B: [Apotelyt S1](https://apotelyt.com/camera-specs/panasonic-s1-sensor-size) -- 35.6 x 23.8 mm

- **Blackmagic BMPCC4K sensor dimensions**: Listed as 18.96 x 10.0 mm by Blackmagic official, which differs slightly from standard MFT (17.3 x 13.0 mm). The BMPCC4K sensor is wider but shorter than a standard MFT sensor due to its video-oriented 17:9 aspect ratio design.
