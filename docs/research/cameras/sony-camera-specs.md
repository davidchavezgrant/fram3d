#### Sony VENICE (MPC-3610)
- **Year**: 2017
- **Sensor Size (W x H mm)**: 36.2 x 24.1 (full sensor); crops to various sizes by mode
- **Sensor Type**: Full-frame CMOS (rolling shutter)
- **Megapixels**: 24.8 MP total (6048 x 4032 max)
- **Supported Resolutions**: Varies by imager mode. Key modes:
  - 6K FF 3:2: 6048 x 4032 (35.9 x 24.0 mm active)
  - 6K FF 17:9: 6054 x 3192 (36.0 x 19.0 mm active)
  - 6K FF 1.85:1: 6054 x 3272 (36.0 x 19.4 mm active)
  - 6K FF 2.39:1: 6048 x 2534 (35.9 x 15.0 mm active)
  - 4K S35 17:9: 4096 x 2160 (24.3 x 12.8 mm active)
  - 4K S35 Anamorphic 4:3: 4096 x 3024 (24.3 x 18.0 mm active)
  - 4K S35 Anamorphic 6:5: 4096 x 3432 (24.3 x 20.4 mm active)
  - UHD 16:9: 3840 x 2160 (22.8 x 12.8 mm active)
  - 2K: 2048 x 1080
- **Lens Mount**: PL mount (native); E-mount (swappable)
- **Native Aspect Ratio(s)**: 3:2, 17:9, 16:9, 1.85:1, 2.39:1, 4:3, 6:5 (depends on mode)
- **Frame Rates**: (with optional HFR license) 6K 3:2 up to 60fps, 6K 17:9 up to 72fps, 6K 2.39:1 up to 90fps, 4K 17:9 up to 110fps, 4K 2.39:1 up to 120fps
- **Sources**: [Sony Pro - VENICE](https://pro.sony/ue_US/products/digital-cinema-cameras/venice), [VFX Camera Database - VENICE](https://vfxcamdb.com/sony-venice/), [The Camera Dept - VENICE](https://www.thecameradept.com/sony-venice)

## Contradictions

### FX6 Sensor Dimensions
- **Newsshooter** and **VideoExpert.eu** report: 35.7 x 18.8 mm (same as FX9, full-frame width but 17:9 vertically cropped)
- **Camera Decision** reports: 36 x 24 mm (standard full-frame)
- **Assessment**: The 35.7 x 18.8 mm figure is more likely correct for the **active imaging area** used for video, as the FX6 is a dedicated cinema camera without a full 3:2 still photo mode. The 36x24 figure may represent the full physical sensor die size. Multiple authoritative cinema sources agree on 35.7 x 18.8 mm.

### NEX-FS700 Sensor Dimensions
- **VFX Camera Database** reports: 24.0 x 12.7 mm
- **Other sources** cite: 23.6 x 13.3 mm (same as FS100/FS5/FS7)
- **Assessment**: The FS700 uses a different, higher-resolution sensor (11.6 MP vs the FS100's lower-res chip), so slightly different dimensions are plausible. The VFX Camera Database is generally reliable for exact sensor dimensions.

### FX6 Sensor -- Full Frame Classification
- The FX6 and FX9 are marketed as "full-frame" cameras, but their sensors measure 35.7 x 18.8 mm, which is vertically cropped compared to a true 36 x 24 mm full-frame sensor. They use the full ~36mm width of a full-frame sensor but only ~19mm height (17:9 aspect). This is a valid marketing classification (the image circle covers full-frame lenses) but the active area is significantly smaller than cameras like the FX3 (35.6 x 23.8 mm) or VENICE (36.2 x 24.1 mm).
