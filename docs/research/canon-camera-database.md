## Contradictions and Notes

### R5 C Video Sensor Active Area
- The R5 C physical sensor is 36.0 x 24.0 mm (same as R5), but in video (cinema) mode, the active recording area is cropped to 17:9 (approximately 36.0 x 19.0 mm for DCI or 33.8 x 19.0 mm for UHD). This is consistent with how Cinema EOS cameras operate -- they use a 17:9 gate for DCI content. The full 3:2 sensor area is only used in photo mode.

### C200 Sensor Dimensions: 24.4 x 13.5 vs 24.6 x 13.8
- Canon's official specifications for the C200 list the active sensor area as 24.4 x 13.5 mm, which is slightly smaller than the C100/C300/C500 original's 24.6 x 13.8 mm. This may reflect a newer sensor design despite being marketed as the same "Super 35mm" category. Both dimensions are correct for their respective cameras.

### C400/C80 Sensor: Full Frame but 17:9
- The C400 and C80 sensor dimensions (36.0 x 19.0 mm) reflect a full-frame width but a narrower height than a traditional full-frame still sensor (36 x 24). This is because cinema cameras use a wider aspect ratio gate. The sensor is "full frame" in width (matching 35mm still film/FF sensors) but cropped vertically to approximately 17:9 for cinema use.

### R6 vs R6 Mark II Sensor Dimensions
- R6: 36.0 x 24.0 mm (apotelyt)
- R6 Mark II: 35.9 x 23.9 mm (apotelyt)
- Minor difference likely reflects different sensor generations. Both are marketed as "full frame."

### Frame Rate Rounding
- Throughout this document, frame rates like "23.98" refer to 23.976 fps, "29.97" to 29.97 fps, "59.94" to 59.94 fps, "119.9" to 119.88 fps, etc. These are the actual NTSC-derived rates. PAL equivalents (25, 50, 100, 200 fps) and true 24.00 fps (for film-out workflows) are also supported where noted.
